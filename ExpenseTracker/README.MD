# 💰 Expense Tracker Web App

A modern and responsive **Expense Tracker** built using **HTML, CSS, and JavaScript**.
Track your income and expenses in real-time with a clean and intuitive UI.

---

## 🚀 Features

- ➕ Add income & expense transactions
- ❌ Delete transactions with confirmation
- 💾 Data saved using **LocalStorage** (persists after refresh)
- 📊 Real-time balance calculation
- 🏷️ Category support (Food, Travel, Shopping, etc.)
- 📅 Date tracking for each transaction
- 🎨 Modern UI with animations and responsive design

---

## 🛠️ Tech Stack

- **HTML5**
- **CSS3 (Flexbox + Grid + Animations)**
- **JavaScript (Vanilla JS)**
- **LocalStorage API**

## ⚙️ How to Use

1. Clone the repository

```
git clone https://github.com/your-username/expense-tracker.git
```

2. Open the project folder
3. Run `index.html` in your browser

---

## 💡 Usage Guide

- Enter a **description** and **amount**
- Use **negative (-)** values for expenses
- Select a category
- Click **Add Transaction**
- View your balance update instantly
